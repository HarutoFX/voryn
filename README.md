# VORYN

VORYN is a React + Vite frontend paired with a Python backend for the workspace and chat experience.

## Local Development

Install frontend dependencies:

```bash
npm install
```

Run the frontend:

```bash
npm run dev
```

The default frontend runs on `http://localhost:3000`.

## Environment

Use `.env.example` as the starting point for local configuration.

## Notes

- `app.html` is the current workspace entry point.
- `index.html` remains available for compatibility while the app is stabilized.
